Unzip the Project and Follow Given Steps to Run the Project
============================================================


------------- Dependent Plugins -------------

Open terminal in unzipped folder and run following commands to install dependencies

Python 3.10 Required....

#In Windows

    venv/Scripts/activate
    pip install -r requirements.txt

#In Linux

    source venv/bin/activate
    pip3 install -r requirements.txt


== Launching the project ==

#For Windows Run Following Command

    cd thingSpeak
    python manage.py runserver

#For Linux Run Following Command

    cd thingSpeak
    python3 manage.py runserver
